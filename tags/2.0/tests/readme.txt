Twease Tests Readme

This directory has a bunch of random files in it used for testing Twease. These are unsupported and are for educational use only.

Twease is developed using Textmate on OS X and uses MTASC for testing. Edit the mtasc.yaml file for the correct classpath for using test.as with MTASC.

Please do not ask questions regarding any of these files, they are unsupported!
